Применительно к Набору Программ по обработке wav-файлов:
  Начало
      Данный проект это набор кода, компиляция оного в программы прозволяет взаимодействующих с wav-файлами.
      Предназначен не только для пользователя, но и программиста - какие шишки были набиты.

      Каждая программа состоит из:
        1. Обработчика передаваемых параметров
        2. Считывания wav-файла (файлов), определяя его (их) особенности.
        3. Преобразование и запись.
        4. Вывод данных пользователю.

  Развитие
      Каждая программа будет содержать:
        1. Парсер параметров
        2. Структуру-класс для хранения характеристик файла, указателей на данные.
        3. Процедуру обработки данных из файла.
        4. Вывод результатов, возможно, логирование.
